Adds a player to the leaderboard.

This function adds a player to the leaderboard, ensuring the leaderboard does not exceed the LEADERBOARD_SIZE constant. If there is space, the new score is added; otherwise, the lowest score is replaced. The leaderboard is then sorted and saved to the file.

Parameters
lbA pointer to the leaderboard structure.
filenameThe name of the file to save the updated leaderboard (without the extension).
nameThe name of the player.
scoreThe score of the player.

Comparison function for sorting player scores.

This function compares two player scores for sorting purposes. It is used by the qsort function to sort players in descending order based on their scores.

Parameters
obj_aA pointer to the first player score to compare.
obj_bA pointer to the second player score to compare.
Returns
A negative integer if obj_a's score is less than obj_b's score, a positive integer if obj_a's score is greater, or 0 if they are equal.
